Flash Card Fun is a dynamic web study application. Once you create a user name and are logged in, this app allows you to store two types of study cards. One is a basic study card which is simply a question on one side and an answer on the other. It also allows you to create cloze study cards. These are fill in the blank cards where one card contains the full answer, and the other side of the card contains the portion you want removed from the question. Using Firebase, the cards created are stored so that you can come back and study anytime once logged in. In addition, cards can be logged by group name so that studying can be done in a more concise manner.
